Provides useful classes which operate on dates and date strings.

Package Specification

The {@link org.neurpheus.core.date.ISO8601Parser} class parses date/time strings written in ISO8601 format. See http://en.wikipedia.org/wiki/ISO_8601 for more informations about the ISO 8601 standard. See http://www.w3.org/TR/NOTE-datetime for informations about a reduced subset of standards supported by this class.
