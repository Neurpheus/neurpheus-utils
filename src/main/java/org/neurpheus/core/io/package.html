Provides classes which help with reading and writing objects.

Package Specification

The {@link org.neurpheus.core.io.Object2FilePath} class writes an object or a collection of objects to the file at a specified location.

The {@link org.neurpheus.core.io.Object2FilePath} class reads an object or a collection of objects from the file stored at a specified location.

These classes encapsulate exceptions related with serialization or other operations (throw only {@link java.io.IOException}), ensure that all used streams will be properly closed, and support compression if required.
